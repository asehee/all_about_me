# Frontend Checklist

Use this checklist before wiring feature logic.

## 1. Publishing First

- [ ] Layout skeleton is finalized.
- [ ] Spacing system is consistent (margin/padding/gaps).
- [ ] Typography scale and emphasis are consistent.
- [ ] Color/mood matches current product direction.

## 2. Responsive

- [ ] Desktop layout verified.
- [ ] Mobile layout verified.
- [ ] Overflow/line wrap issues checked.

## 3. Interaction

- [ ] Hover, focus, and active states are designed.
- [ ] Button/input/modal states are consistent.
- [ ] Transition/animation speed is not distracting.

## 4. Wiring

- [ ] Local state and events are connected after publishing sign-off.
- [ ] API calls are connected after UI baseline approval.
- [ ] Error/loading/empty states are represented in UI.

## 5. Final QA

- [ ] Pixel alignment and baseline checks completed.
- [ ] Build passes.
- [ ] Regression check on related pages completed.
